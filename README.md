# Secure_ORAN
# Work in Progress!

The rising demand for IoT devices in healthcare has
increased the need for intelligent communication frameworks
which are capable of supporting secure information exchange,
while ensuring low latency data transmission. While DRL has
been used for adaptive traffic management, the existing approach
primarily focuses on communication efficiency and overlooks
data security during transmission. This paper proposes a Secure
DRL ORAN framework that integrates dynamic matrix encryp-
tion with PPO based routing. The results demonstrate that the
proposed framework achieves packet integrity of 97.96% with
latency reduction of 16.44% compared to conventional routing
methods.
